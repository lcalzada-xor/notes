# Bug Bounty Notes & Pipelines

This repository contains a collection of notes, templates, and pipelines for Bug Bounty hunting.

## Structure

- **`gftemplates/`**: Custom GF patterns for finding vulnerabilities and interesting data in URLs.
- **`pipelines/`**: Step-by-step guides and command-line workflows for reconnaissance and vulnerability scanning.
- **`regex/`**: Useful regex patterns for various tasks.

## Usage

Navigate to the specific directory for the resource you need. The `pipelines` directory contains Markdown files with commands that can be copied and adapted to your target.
